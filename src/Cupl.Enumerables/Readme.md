Cupl.Enumerables is a Cupl library extending Enumerable, Collection, and String types, with a focus on working with nullables.

`Cupl.Enumerables.EnumerableUtils` offers extension methods similarly to `System.Linq.Enumerable`.

`Cupl.Enumerables.CollectionUtils` offers extension methods to various collection types, such as `IReadOnlyDictionary<,>`, `IList<>`, `Queue`, and `Stack`.

`Cupl.Enumerables.StringUtils` offers extension methods to `String`.
